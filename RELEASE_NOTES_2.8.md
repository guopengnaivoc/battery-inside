## 电池内显 2.8

- 电池内部填充柱现在会随电量百分比连续缩短
- 使用 `20.8 pt × 百分比` 计算宽度；每 1% 约为 `0.208 pt`
- 使用 Core Graphics 亚像素绘制，不要求电池内部拥有100个整数像素
- 数字继续显示精确百分比，填充柱用于直观表达剩余电量
- 改善深色菜单栏对比度：文字、闪电和插头在填充与空白区域自动使用合适颜色
- 保持原有颜色阈值：30%及以上白色、10%–29%橘黄色、9%及以下红色
- 同步更新五语言说明、真实菜单栏图片和颜色规则图

支持 macOS 13+、Apple 芯片和 Intel Mac。当前安装包尚未经过 Apple Developer ID 公证。
