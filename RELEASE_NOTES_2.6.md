## 电池内显 2.6

首次公开发布。

- 电池内部显示百分比，并随剩余电量改变填充长度
- 30% 及以上白色、10%–29% 橘黄色、9% 及以下红色
- 充电显示闪电，接通电源但未充电显示插头
- 登录时自动启动、低电量通知、设置窗口和安全卸载
- 支持 macOS 13+、Apple 芯片和 Intel Mac

当前安装包尚未经过 Apple Developer ID 公证。首次打开的方法请参阅仓库 README 或 DMG 内的《安装与使用说明》。
