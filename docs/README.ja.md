# BatteryInside（電池内表示）

[中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · [Français](README.fr.md) · [Italiano](README.it.md)

![メニューバー表示のプレビュー](images/hero.svg)

BatteryInside は、バッテリー残量、パーセント、電源状態を1つのコンパクトなアイコン内に表示する、軽量で読み取り専用の macOS メニューバーアプリです。

作者：郭鹏（Guo Peng）

## 3ステップでインストール

![DMGをダウンロードし、アプリケーションへドラッグして開く](images/install.svg)

1. [Releases](/guopengnaivoc/battery-inside/releases/latest) から最新の `BatteryInside-バージョン.dmg` をダウンロードします。
2. DMGを開き、BatteryInsideを「アプリケーション」へドラッグします。
3. Finder → アプリケーションから BatteryInside を開きます。メニューバーに表示されます。

### 初回起動がmacOSにブロックされた場合

現在の公開版はアドホック署名で、Apple Developer IDによる公証は行われていません。「開発元を確認できない」などと表示された場合：

1. 一度アプリを開き、警告を閉じます。
2. システム設定 → プライバシーとセキュリティを開きます。
3. セキュリティ欄の BatteryInside に関する表示から「このまま開く」を選びます。

本プロジェクトの GitHub Release から入手し、SHA-256 が一致するファイルだけにこの操作を行ってください。Gatekeeper 全体を無効にしないでください。

## 状態表示

![バッテリーの色と電源状態](images/status.svg)

- 30%以上：内部全体が白
- 10%～29%：内部全体がオレンジ
- 9%以下：内部全体が赤
- 充電中：稲妻
- 電源接続中・非充電：プラグ
- 情報を取得できない場合：`--`

現在のバージョンでは、残量に応じて塗りつぶしの長さは短くなりません。警告色が変わるのは内部全体だけです。外枠と右側の端子は macOS の `labelColor` に従い、オレンジや赤にはなりません。数字、稲妻、プラグは常に黒です。

電源状態の判断には、macOS が明示する `Is Charging`、`Power Source State`、`Is Charged` のみを使用します。

## 設定とシステムアイコンの置き換え

![設定を開き、Appleのバッテリーアイコンを非表示にする](images/settings.svg)

メニューバーの表示は読み取り専用で、クリックしても反応しません。設定を変更するには Finder → アプリケーションから BatteryInside をもう一度開きます。ログイン時の起動、20%・10%の低残量通知、終了、安全なアンインストールを設定できます。

Apple標準のアイコンを隠すには：

- 新しいmacOS：システム設定 → メニューバー → メニューバーのコントロール → バッテリー
- macOS 13～15：システム設定 → コントロールセンター → バッテリー →「メニューバーに表示」をオフ

macOS のバッテリー機能自体は削除・変更されません。同じ場所で再度オンにすると復元できます。

## 動作環境とプライバシー

- macOS 13以降
- Appleシリコン／Intel Mac
- ネットワーク通信、解析、データ収集なし
