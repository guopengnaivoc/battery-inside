# BatteryInside（電池内表示）

[中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · [Français](README.fr.md) · [Italiano](README.it.md)

![メニューバー表示のプレビュー](images/hero.svg)

BatteryInside は、バッテリー残量、パーセント、電源状態を1つのコンパクトなアイコン内に表示する、軽量で読み取り専用の macOS メニューバーアプリです。

作者：郭鹏（Guo Peng）

## 3ステップでインストール

![DMGをダウンロードし、アプリケーションへドラッグして開く](images/install.svg)

1. [Releases](/guopengnaivoc/battery-inside/releases/latest) から最新の `BatteryInside-バージョン.dmg` をダウンロードします。
2. DMGを開き、BatteryInsideを「アプリケーション」へドラッグします。
3. Finder → アプリケーションから BatteryInside を開きます。メニューバーに表示されます。

> アップデート時は先に BatteryInside を終了し、新版を「アプリケーション」へドラッグして「置き換える」を選びます。2.8.4 では、一部の Intel Mac で残る Launchpad の古いアイコンキャッシュを避けるため、アイコンリソース名を変更しました。Finder は角丸なのに Launchpad が一時的に四角い場合は、Mac を一度再起動してください。

### 初回起動がmacOSにブロックされた場合

![未署名アプリを安全に開く：ゴミ箱へ移動せず、プライバシーとセキュリティからこのまま開く](images/open-anyway.svg)

現在の公開版はアドホック署名で、Apple Developer IDによる公証は行われていません。「開発元を確認できない」などと表示された場合：

1. 一度アプリを開き、警告を閉じます。
2. システム設定 → プライバシーとセキュリティを開きます。
3. セキュリティ欄の BatteryInside に関する表示から「このまま開く」を選びます。

本プロジェクトの GitHub Release から入手し、SHA-256 が一致するファイルだけにこの操作を行ってください。Gatekeeper 全体を無効にしないでください。

「コンピュータに損害を与える」、マルウェア検出、またはファイル破損と明示された場合は操作を中止して再ダウンロードし、「このまま開く」を選ばないでください。このボタンは通常、最初の起動試行から約1時間だけ表示されます。

参考：[Apple公式のセキュリティ設定を上書きしてアプリを開くガイド](https://support.apple.com/guide/mac-help/open-an-app-by-overriding-security-settings-mh40617/mac)。

## 状態表示

![バッテリーの色と電源状態](images/status.svg)

- 30%以上：白い残量バー
- 10%～29%：オレンジの残量バー
- 9%以下：赤い残量バー
- 充電中：稲妻
- 電源接続中・非充電：プラグ
- 情報を取得できない場合：`--`

残量バーの幅は `20.8 pt × パーセント` で連続的に変化します。1%は約 `0.208 pt` で、Core Graphics のサブピクセル描画を使うため、100個の整数ピクセルは必要ありません。数字が正確な値を、バーが視覚的な目安を示します。外枠と端子は macOS の `labelColor` に従い、文字と電源記号は塗りつぶし部分では黒、空の部分ではシステム色になります。

電源状態の判断には、macOS が明示する `Is Charging`、`Power Source State`、`Is Charged` のみを使用します。

## 設定とシステムアイコンの置き換え

![設定を開き、Appleのバッテリーアイコンを非表示にする](images/settings.svg)

メニューバーの表示は読み取り専用で、クリックしても反応しません。設定を変更するには Finder → アプリケーションから BatteryInside をもう一度開きます。ログイン時の起動、20%・10%の低残量通知、終了、安全なアンインストールを設定できます。

ログイン時の起動は静かに行われます。再起動または再ログイン後はメニューバーのバッテリーだけが表示され、設定ウインドウは開きません。設定が必要なときに Finder → アプリケーションから BatteryInside を再度開いてください。

Apple標準のアイコンを隠すには：

- 新しいmacOS：システム設定 → メニューバー → メニューバーのコントロール → バッテリー
- macOS 13～15：システム設定 → コントロールセンター → バッテリー →「メニューバーに表示」をオフ

macOS のバッテリー機能自体は削除・変更されません。同じ場所で再度オンにすると復元できます。

### 他のアプリアイコンの右端に配置する

![Commandキーを押しながらBatteryInsideをドラッグ](images/position.svg)

1. `Command（⌘）`キーを押したままにします。
2. キーを押したまま、メニューバーの BatteryInside をマウスまたはトラックパッドでドラッグします。
3. 他のサードパーティ製アプリアイコンの右側で離します。

BatteryInside には固定のメニューバー保存名が設定されているため、アプリの再起動、Mac の再起動、アップデート後も macOS が選択した位置を復元します。各 Mac への初回インストール後に一度だけ行ってください。時計、コントロールセンター、プライバシー表示はシステム予約領域にあり、その右側へアプリを移動することはできません。

## 動作環境とプライバシー

- macOS 13以降
- Appleシリコン／Intel Mac
