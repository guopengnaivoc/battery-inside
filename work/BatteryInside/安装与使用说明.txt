电池内显 2.8.4｜安装与使用说明
作者：郭鹏

本 DMG 还包含“使用说明 · Guide.html”。双击即可查看带图片的中文、英文、日文、法文和意大利文说明。

一、安装

1. 将“电池内显”拖到右侧的“应用程序”文件夹。
2. 打开 Finder，在“应用程序”中双击“电池内显”。
3. 启动后，电池百分比会显示在屏幕顶部菜单栏中。

二、如果提示“无法验证开发者”或“Apple 无法检查是否包含恶意软件”

本版本没有经过 Apple Developer ID 公证，因此首次打开时可能被 macOS 阻止。
只有在确认安装包来自本项目官方 GitHub 发布页面时，才继续以下操作：

1. 先在“应用程序”中双击一次“电池内显”。如果警告提供“移到废纸篓”和“完成”，请点击“完成”，不要点击“移到废纸篓”。
2. 打开“系统设置”→“隐私与安全性”。
3. 向下滚动到“安全性”，找到关于“电池内显”被阻止的提示。
4. 点击“仍要打开”。该按钮通常只在首次尝试启动后的约一小时内显示。
5. 输入 Mac 登录密码，再次确认“打开”。

以后可以像普通应用一样打开。如果新版本再次出现提示，重复以上步骤即可。
部分 macOS 版本也可以在 Finder 中按住 Control 点击“电池内显”，选择“打开”。

不要关闭整个 Gatekeeper，也不要运行来源不明的终端解除命令。
如果系统明确提示应用“会损坏电脑”或检测到恶意软件，请不要强行打开，并重新核对下载来源。

三、用“电池内显”替代系统自带的菜单栏电池图标

“电池内显”不会删除或修改 macOS 的电池功能。你可以只隐藏系统自带的菜单栏电池图标，
在视觉上由“电池内显”替代它；系统设置、电池健康和控制中心内的电池功能仍然保留。

较新版本的 macOS：

1. 打开“系统设置”。
2. 点击侧边栏中的“菜单栏”（可能需要向下滚动）。
3. 在“菜单栏控制项”中找到“电池”。
4. 关闭或取消选择“电池”的菜单栏显示。

macOS 13、14 或 15：

1. 打开“系统设置”。
2. 点击侧边栏中的“控制中心”。
3. 找到“电池”。
4. 关闭“在菜单栏中显示”。

如果以后想恢复系统电池图标，回到同一位置重新开启即可。

四、把“电池内显”放在其他应用图标的最右侧

1. 按住键盘上的 Command（⌘）键。
2. 保持按住 Command，用鼠标或触控板拖动菜单栏中的“电池内显”。
3. 把它放到其他第三方应用图标的右侧，然后松开。

应用会通过 macOS 的菜单栏位置记忆功能保存这个位置。以后重新打开、重启电脑或升级应用时，
系统会尽量恢复它。每台新电脑首次安装后需要操作一次。时间、控制中心和隐私指示器等系统项目
拥有 macOS 保留位置，“电池内显”不能移动到这些系统项目的右侧。

五、设置、退出和卸载

菜单栏中的“电池内显”是只读指示器，点击它不会弹出菜单。

如需设置：在 Finder 的“应用程序”文件夹中再次打开“电池内显”。设置窗口提供：

• 登录时自动启动
• 低电量通知（20% 和 10%）
• 退出应用
• 安全卸载（应用会移到废纸篓并清除自身设置）

六、颜色和状态

• 30% 及以上：白色填充柱
• 10% 至 29%：橘黄色填充柱
• 9% 及以下：红色填充柱
• 正在充电：显示闪电
• 已接通电源但没有充电：显示插头
• 无法读取电池信息：显示“--”

填充柱长度会按照“20.8 pt × 电量百分比”连续变化。每 1% 约等于 0.208 pt，
通过 Core Graphics 亚像素绘制，不需要电池内部拥有100个整数像素。数字负责显示精确电量，填充柱负责直观显示剩余量。
外框和右侧电池头跟随 macOS 系统颜色；数字和供电符号会自动适配已填充与未填充区域的明暗对比。

七、隐私与兼容性

电池内显仅在本机读取 macOS 提供的电池状态，不联网，不收集或上传数据。
支持 macOS 13 及以上版本，同时支持 Apple 芯片和 Intel Mac。

八、Intel Mac 或“启动台”仍显示旧的方形图标

2.8.4 已更换应用内部的图标资源名称，用于绕开 macOS 对旧图标的缓存。请先退出旧版，
再把新版拖入“应用程序”并选择替换。若 Finder 已显示圆角、但“启动台”仍短暂显示旧图标，
请重启 Mac；系统重新建立应用列表后会读取新版圆角图标。
